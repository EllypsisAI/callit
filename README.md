# CallIt

Play-money prediction platform for content creators' communities.

**Status:** Phase 1 — Research

## What is this?

Fans predict outcomes in creator content (FIFA tournaments, challenges, competitions) using points called "calls." No real money. No gambling. Just entertainment software that makes content more engaging.

## Proof of concept

Built around [CREW](https://youtube.com/@crewboyss) — a Danish YouTube group — and their FIFA tournament.
